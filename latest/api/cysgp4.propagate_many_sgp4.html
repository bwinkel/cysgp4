propagate_many_sgp4¶

cysgp4.propagate_many_sgp4(mjds, tles, observers=None, do_eci_pos=True, do_eci_vel=True)[source] [edit on github]¶

This is an sgp4-based variant (non-parallelized, Python-looping) of propagate_many that is meant for testing and benchmarking only. It has (almost) the same interface, except that do_geo, do_topo, and on_error are not handled.

Note: sgp4 Package by Brandon Rhodes
